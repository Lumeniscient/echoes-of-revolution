# Echoes of the Revolution
## Heroes of Negros Island Region

An AI-powered historical detective game
built with Godot Engine 4 for 
AI Game On! 2026 — AI Fest Philippines.

### Team Triple X
- Mico Abrasaldo
- Ron Gabriel Della
- Mark Laurence Madreña

### About the Game
Echoes of the Revolution is a Criminal 
Case-inspired detective game where players 
investigate the mystery of Papa Isio —
was he a bandit, a rebel, or a freedom 
fighter?

### Technology
- Godot Engine 4.x (GDScript)
- Google Gemini API (AI dialogue)
- AI-generated art assets

### How to Play
1. Download the latest release
2. Run EchoesOfTheRevolution.exe
3. If antivirus flags it, click 
   "Run Anyway" — false positive
   common with Godot games

### Features
- Hidden object investigation gameplay
- AI-powered witness dialogue
- Memory Conflict detection system
- Hidden Heritage Artifact mechanic
- Historical facts about Papa Isio
